# スライドショー自動生成ツール

このツールは、指定したフォルダに入れた画像とBGMから、自動的に「Ken Burns効果（ズームやパン）」のついたスライドショー動画（MP4）を生成するPythonプログラムです。
FFmpegを内部で使用して動画を作成します。

## フォルダ構成
このツールがあるフォルダ構成は以下のようになります。

```text
auto_slideshow_tool/
├── generate_video.py  (実行ファイル)
├── README.md          (この説明書)
├── images/            (画像を入れるフォルダ)
│   ├── 01.jpg         (※アルファベット順または数字順に処理されます)
│   ├── 02.png
│   └── ...
└── bgm/               (音楽を入れるフォルダ)
    └── BGM.mp3        (※1ファイルのみ使用されます)
```

## 使い方

1. **準備:** 
   - `images` フォルダにお好きな画像ファイル（.jpg, .png など）を入れます。
     ※ファイル名の順番（01.jpg, 02.jpg など）でスライドショーが構成されます。
   - `bgm` フォルダにお好きな音楽ファイル（.mp3 または .wav）を1つ入れます。
     ※BGMを入れなかった場合は、無音のスライドショーが作成されます。

2. **実行:**
   コマンドプロンプトやPowerShell等でこのディレクトリに移動し、以下のコマンドを実行します。

   ```bash
   cd c:\Users\24tom\workspace\hideaki\auto_slideshow_tool
   python generate_video.py
   ```

3. **完了:**
   処理が完了すると、同じフォルダ内に `slideshow.mp4` という名前の動画が出力されます！

## 設定の変更について
`generate_video.py` をテキストエディターで開くと、上部にある設定値を簡単に変更できます。

- `duration_per_image = 5` : 1枚の画像あたりの表示秒数（初期設定は5秒）
- `fps = 30` : 動画のフレームレート（滑らかさ）

## 注意事項
- PCに `ffmpeg` がインストールされており、環境変数（Path）が通っている必要があります。
- 画像サイズがバラバラでも、内部で自動的に 1920x1080 に調整して黒帯などを補って処理します。
- 生成完了までにPCのスペックによっては数分かかる場合があります。
